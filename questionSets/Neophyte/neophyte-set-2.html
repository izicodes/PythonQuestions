Neophyte Set #2

There are 15 questions. Mix of multiple-choice, code creation and justification questions.

Topics: Variables, Expressions, Arithmetics, Conditional Statements, Loop Statements

  1. Which of the following is a valid way to declare a variable in Python?

    1. variable == 5
    2. 5 = variable
    3. variable = 5
    4. variable := 5

  2. What is the result of the following expression?

    x = 10
    y = 3
    result = x % y

    1. 3
    2. 1
    3. 0
    4. 2

  3. Which statement is used to execute a block of code repeatedly as long as a certain condition is true?

    1. for statement
    2. elif statement
    3. if statement
    4. while statement

  4. Write a Python code snippet to declare a variable "age" and assign it the value 25.

  5. Write a Python code snippet to calculate the average of three numbers: 10, 15, and 20. Display the result using the print function.

  6. Write a Python code snippet using a while loop to print the numbers from 1 to 10.

  7. Explain the difference between the if statement and the elif statement in Python.

  8. Describe the purpose of the range() function in Python.

  9. How does the break statement work in a loop in Python?

  10. Explain the concept of operator precedence in Python.

  11. Write a Python code snippet that prompts the user to enter their name and age. Based on the age entered, display one of the following messages:

    If the age is less than 18, display "You are a minor."
    If the age is between 18 and 65 (inclusive), display "You are an adult."
    If the age is greater than 65, display "You are a senior citizen."

  12. Identify and correct the error(s) in the following code snippet:

    x = 5
    y = "3"
    result = x + y
    print(result)

  13. Identify and correct the error(s) in the following code snippet:

    for i in range(1, 6)
    print(i)

  14. Identify and correct the error(s) in the following code snippet:

    age = input("Enter your age: ")
    if age < 18:
    print("You are not eligible.")
    else:
    print("You are eligible.")

  15. Identify and correct the error(s) in the following code snippet:

    while True:
    print("Hello, world!")
