Neophyte Set #3

There are 15 questions. Write the code to solve the following problems.

Topics: Variables, Expressions, Arithmetics, Conditional Statements, Loop Statements

  1. Write a program that takes an integer input from the user and prints "Positive" if the number is greater than zero, "Negative" if the number is less than zero, and "Zero" if the number is equal to zero.

  2. Write a program that takes a year as input and determines whether it is a leap year or not. (A leap year is divisible by 4 but not divisible by 100, except if it is divisible by 400.)

  3. Write a program that takes two numbers as input and prints the larger of the two numbers.

  4. Write a program that takes three numbers as input and prints the smallest number.

  5. Write a program that takes an integer input from the user and checks if it is a prime number.Write a Python code snippet to calculate the average of three numbers: 10, 15, and 20. Display the result using the print function.

  6. Write a program that prints all the even numbers between 1 and 50.

  7. Write a program that calculates the sum of all the numbers between 1 and 100.

  8. Write a program that takes a positive integer input from the user and prints the factorial of that number. (Factorial of 5 is 5! = 5 x 4 x 3 x 2 x 1)

  9. Write a program that takes a positive integer input from the user and prints the Fibonacci sequence up to that number. (The Fibonacci sequence starts with 0 and 1, and each subsequent number is the sum of the two previous numbers.)

  10. Write a program that counts the number of vowels in a given string.

  11. Write a program that takes a positive integer input from the user and prints the reverse of the number. (For example, if the input is 12345, the output should be 54321)

  12. Write a program that takes a positive integer input from the user and checks if it is a palindrome. (A palindrome is a number that reads the same backward as forward. For example, 121 is a palindrome.)

  13. Write a program that takes a positive integer input from the user and prints the sum of its digits. (For example, if the input is 123, the output should be 1 + 2 + 3 = 6)

  14. Write a program that prints all the prime numbers between 1 and 100.

  15. Write a program that prints the following pattern using nested loops:

    *
    **
    ***
    ****
    *****

End of Exam
